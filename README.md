# Compulsive Behavior Block Lists

## Overview

This repository contains curated block lists designed to help individuals manage and reduce three interconnected forms of compulsive consumption: **shopping addiction**, **gear acquisition syndrome**, and **media consumption addiction**. These lists target websites, apps, and services that enable and encourage excessive purchasing and media consumption patterns.

## Purpose

These block lists serve as practical tools for:

- **Reducing temptation** by blocking access to known platforms during moments of weakness
- **Creating friction** in compulsive browsing and purchasing patterns to allow time for reflection
- **Supporting recovery** for individuals working to develop healthier consumption habits
- **Breaking cycles** of habitual behavior and algorithmic manipulation
- **Protecting mental and financial health** from the designed addictiveness of these platforms

## How to Use

These block lists can be integrated with:
- Browser extensions (ad/content blockers like uBlock Origin, Freedom, LeechBlock)
- Router-level DNS filtering (Pi-hole, NextDNS)
- App blockers on mobile devices (Freedom, Digital Wellbeing, AppBlock)
- Focus and productivity tools (Cold Turkey, Focus@Will)
- Operating system-level hosts file modifications

## Important Note

These block lists are intended as **supportive tools for self-improvement**, not as a complete solution. They work best as part of a comprehensive approach that includes:

### Shopping & GAS Recovery
- Creating and maintaining a budget
- Addressing underlying emotional triggers (anxiety, loneliness, boredom)
- Practicing mindful consumption and the "waiting rule" before purchases
- Seeking professional financial or psychological support if needed
- Building healthier hobbies that don't center on acquisition

### Media Consumption Recovery
- Establishing healthy screen time boundaries and "no-phone" times
- Cultivating offline hobbies and face-to-face relationships
- Practicing mindfulness and digital wellness
- Improving sleep hygiene by reducing evening screen time
- Seeking professional mental health support if needed
- Understanding and resisting algorithmic manipulation tactics
